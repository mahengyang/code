快拿推荐评测（app）
===============
item-cf
-------------

#### 说明

**训练集/测试集数据格式**
: 10000001 => {QQ，微信}

**关联矩阵格式**
: QQ => {微信 => 0.9，快拿 => 0.8} 
: 按关联度从高到低排序，保留前五个

**推荐方法**
: 从关联矩阵中查找用户的每一个共享所关联的共享，合并去重后做为其推荐

**命中次数**
: 推荐结果集与用户已有的共享集的交集大小
